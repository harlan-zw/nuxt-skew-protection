A headless Vue component that provides notification logic without imposing any UI.

## Basic Usage

```vue
<template>
  <SkewNotification v-slot="{ isOpen, dismiss, reload, timeAgo }">
    <div v-if="isOpen" class="notification">
      <p>New version available! ({{ timeAgo }})</p>
      <button @click="reload">
        Reload
      </button>
      <button @click="dismiss">
        Dismiss
      </button>
    </div>
  </SkewNotification>
</template>
```

## Slot Props

The component provides these props via scoped slot:

### `isOpen`

- Type: `boolean`

Whether the notification should be shown.

Automatically set to `true` when:

- User's loaded modules are invalidated by a new deployment
- Can be manually controlled via the `open` prop

```vue
<SkewNotification v-slot="{ isOpen }">
  <div v-if="isOpen">
    Update available!
  </div>
</SkewNotification>
```

### `dismiss()`

- Type: `() => void`

Dismiss the notification (sets `isOpen` to `false`).

```vue
<SkewNotification v-slot="{ isOpen, dismiss }">
  <div v-if="isOpen">
    <p>Update available!</p>
    <button @click="dismiss">Not now</button>
  </div>
</SkewNotification>
```

### `reload()`

- Type: `() => void`

Reload the page to get the new version.

Calls `window.location.reload()`.

```vue
<SkewNotification v-slot="{ isOpen, reload }">
  <div v-if="isOpen">
    <button @click="reload">Update Now</button>
  </div>
</SkewNotification>
```

### `timeAgo`

- Type: `string`

Human-readable time since the release.

Examples: "just now", "2 minutes ago", "1 hour ago"

```vue
<SkewNotification v-slot="{ isOpen, timeAgo }">
  <div v-if="isOpen">
    <p>Released {{ timeAgo }}</p>
  </div>
</SkewNotification>
```

### `releaseDate`

- Type: `Date | null`

The release date as a Date object.

```vue
<SkewNotification v-slot="{ isOpen, releaseDate }">
  <div v-if="isOpen">
    <p>Released: {{ releaseDate?.toLocaleString() }}</p>
  </div>
</SkewNotification>
```

## Props

### `open`

- Type: `boolean`
- Default: Controlled automatically

Manually control the open state.

```vue
<script setup>
const forceOpen = ref(true)
</script>

<template>
  <SkewNotification v-slot="{ reload }" :open="forceOpen">
    <div>
      <button @click="reload">
        Update
      </button>
    </div>
  </SkewNotification>
</template>
```

### With Auto-Reload Countdown

```vue
<script setup>
const countdown = ref<number | null>(null)
let countdownInterval: NodeJS.Timeout | null = null

function startCountdown(reload: () => void) {
  countdown.value = 10

  countdownInterval = setInterval(() => {
    if (countdown.value === null)
      return

    countdown.value--

    if (countdown.value <= 0) {
      reload()
      if (countdownInterval)
        clearInterval(countdownInterval)
    }
  }, 1000)
}

function stopCountdown() {
  if (countdownInterval) {
    clearInterval(countdownInterval)
    countdownInterval = null
  }
  countdown.value = null
}

onUnmounted(() => {
  stopCountdown()
})
</script>

<template>
  <SkewNotification v-slot="{ isOpen, reload, dismiss }">
    <div v-if="isOpen" class="notification">
      <p>New version available!</p>

      <div v-if="countdown === null">
        <button @click="() => { startCountdown(reload) }">
          Auto-reload in 10s
        </button>
        <button @click="dismiss">
          Dismiss
        </button>
      </div>

      <div v-else>
        <p>Reloading in {{ countdown }}s...</p>
        <button @click="reload">
          Reload Now
        </button>
        <button @click="stopCountdown">
          Cancel
        </button>
      </div>
    </div>
  </SkewNotification>
</template>
```

## Behavior

The component automatically shows when:

1. A new version is deployed
2. The user's loaded modules are deleted in the new version (intelligent detection)

The component automatically hides when:

- User clicks dismiss
- User clicks reload (page reloads)
- `open` prop is set to `false`
