## Skew Cookie

The module stores the user's current build version in a cookie named `__nkpv`. See [cookie options](/docs/skew-protection/api/config#cookie) for full details.

Update your cookie consent settings to allow this cookie.

## Cookie Consent Requirements

> Cookie Name: `__nkpv`
> Purpose: Store the user's current build version for Nuxt Skew Protection.
> Duration: Session (deleted when the browser is closed).
> Type: Functional Cookie
